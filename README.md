# ue-l204
Mini-projet de l'UE-L204

## Base de données

Voici une modélisation de la base de données pour ce mini-projet. IL s'agit d'une BDD à destination d'une université
qui permettrait de gérer des étudiants et des enseignants autour de cours.

![Diagramme de la BBD](docs/svg/UE_L204_mini-Projet_DB.svg)

## Pour initialiser la base de données

Exécuter les scripts suivant **en respectant l'ordre** !

1. `universiteDB_access.sql`
2. `universiteDB_tables.sql`
3. `universiteDB_populate.sql`

Tous les mots de passe sont `123456` par défaut, y compris lors de création de nouvel utilisateur dans la base.
