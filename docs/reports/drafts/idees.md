# Notes / Idées

## Rôles

- Prof : 
  - il peut créer des cours (/!\ penser à gérer les prérequis lors de la création du cours) ;
  - lister les cours ;
  - lister les élèves inscrits dans un cours ;
  - valider un élève sur un cours dont il est responsable.
- Élève :
  - il peut s'inscrire à un cours (validation en fonction des prérequis) : typiquement, dans la liste des cours qu'il liste, ils y a des cases à cocher devant les cours auxquels il peut prétendre et les autres sont désactivés ;
  - lister des cours pour voir leur description, le nb d'inscrit, etc ;
- Admin :
  - il peut créer/supprimer des utilisateurs de type prof et de type élèves (/!\ penser que la création se fait en deux temps : la création de l'utilisateur générique -> puis avec l'ID on spécialise cet utilisateur) ;
  - tout lister ;
